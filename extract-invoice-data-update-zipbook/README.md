**Invoice extraction sample**

This sample process makes use of the read pdf functionality and uses string manipulation to extract the required data.
This may not the perfect/right solution always, but if you are a beginner, this will give you an idea on how to extract pdf data and use string manipulation.
please get in touch with me @ sibi.luke@quadance.com if you have any queries.

**Please ensure the following**

1. chrome add in needs to be installed
2. You will need to assign values to the following parameters. 
	UserName – Username for Zipbooks application. (You need to register first at https://app.zipbooks.com/.)
	Password – Password for Zipbooks application
	InputFilePath – In this location, the user can keep one or more pdf files
	OutputFilePath – All the outputs will be saved in this location
